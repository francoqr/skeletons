## npm stuff

Core

Use supplied packaje.json or follow something like the following:

````bash
npm init
npm install express morgan
npm install -D nodemon
npm install -D typescript @types/node ts-node
npx gitignore Node
git init
```

## IDE (Sublime Text) Help

Keep the already included jsconfig.js to trigger the Language Server:
```json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "ESNext"
  }
}
```
